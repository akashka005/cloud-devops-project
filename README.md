# QN-1

## 1. Initialize Git Repository
- Initialized a new Git repository using:
  git init

## 2. Add Files to Staging Area
- Added project files to staging area:
  git add .

## 3. Commit Initial Project
- Committed files with message:
  git commit -m "initial project setup"

## 4. Create New Branch
- Created a new branch named feature-login:
  git checkout -b feature-login

## 5. Add New File
- Created a file named login.txt:
  touch login.txt

## 6. Stage and Commit File
- Added file to staging:
  git add login.txt
- Committed changes:
  git commit -m "added login.txt file in feature-login branch"
